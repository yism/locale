# Support

## Questions and implementation help

Use GitHub Discussions for design questions, integration help, and adoption
guidance once the public repository is live:

- https://github.com/yism/locale/discussions

## Bug reports

Use GitHub Issues for reproducible bugs, transcript drift, chronology mismatch,
host compatibility regressions, and docs drift:

- https://github.com/yism/locale/issues

Please use the issue templates so reports include the right truth artifacts.

## Security issues

For vulnerabilities or trust-boundary concerns, follow [SECURITY.md](./SECURITY.md)
instead of opening a public issue.

## What maintainers can help with

- MCP and A2A integration questions
- capability issuance and offline verification flows
- transcript replay and host profile validation
- public contract drift and chronology mismatches

## What this repo does not support

- downstream tool execution
- secret brokerage
- payment handling
- general workflow orchestration
